# Contributing

Thank you for your interest in AetherDesk Mobile. 

Please be advised that **no contributions are invited at this time**.

This project was engineered, designed, and built strictly for **personal usage only**. It is not structured to accept Pull Requests, issues, or external collaborative input. 

If you have stumbled across this repository, you may review the code for personal educational purposes, but please refrain from submitting Pull Requests or attempting to fork and modify the codebase for external distribution. Refer to the [Code of Conduct](CODE_OF_CONDUCT.md) for detailed copyright and usage restrictions.

Thank you for your understanding and respect for these boundaries.
