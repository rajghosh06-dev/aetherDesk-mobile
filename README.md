# AetherDesk Mobile

![AetherDesk](app/src/main/assets/www/images/logo.png)

AetherDesk Mobile is a powerful, hybrid Android application that delivers a premium document scanning and management experience. Designed to be completely offline and lightning-fast, it leverages Google's ML Kit Document Scanner to provide hardware-accelerated cropping, filtering, and edge detection.

With its breathtaking **Liquid Glass Frost** theme seamlessly adapting to both Light and Dark modes, AetherDesk Mobile is built not just for utility, but for a flawless user experience.

## ✨ Features
- **High-Fidelity Document Scanning**: Hardware-accelerated scanning using [Google ML Kit Document Scanner API](https://developers.google.com/ml-kit/vision/doc-scanner).
- **Intelligent Memory Management**: Strict 38-page batch limits to guarantee smooth, crash-free operation on all devices.
- **Native Android Bridging**: Exports files to PDF and Image with exact Blob sizing and seamlessly hands them off to native system intents (WhatsApp, Google Drive, OneDrive).
- **Responsive UI/UX**: Built with a visually stunning, responsive hybrid web-layer featuring silky-smooth SortableJS drag-and-drop mechanics.
- **System-level Notifications**: Real-time native Android notifications upon successful export (Android 13+ compliant).
